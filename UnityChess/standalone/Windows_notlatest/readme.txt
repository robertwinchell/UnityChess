Software for free use. Not commercial.

Created by Chessforeva (http://chessforeva.appspot.com) 

Were used:

1) free licence version of Unity3D game development platform
   (http://www.unity3d.com);

2) Valil chess engine C# code (http://www.valil.com);

3) Free models of pieces on internet (thanks to authors);

4) OpenSource Blender modeling software (http://www.blender.org);

5) Javascript chess code libraries (http://chessforeva.blogspot.com);

6) Crafty chess engine (http://www.craftychess.com),
   Rybka (http://www.rybkachess.com),
   Stockfish (http://www.stockfishchess.com) added for strong chess play.

7) Connected to FICS (freechess.org) chess server via PuTTY telnet.
